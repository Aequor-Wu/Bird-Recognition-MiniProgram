# 项目总体概述（Project Overview）

## 1. 项目背景与研究意义

随着移动互联网与人工智能技术的发展，
基于深度学习的图像识别技术在智能应用中得到了广泛应用。
鸟类识别作为典型的**细粒度图像分类（Fine-grained Image Classification）**任务，
在生态保护、物种调查、自然教育等领域具有实际应用价值。

然而，不同鸟类物种在整体外观结构上高度相似，
而类别区分往往依赖于羽毛纹理、颜色分布、喙部形态等局部特征，
这对模型的特征提取能力提出了较高要求。
因此，如何在保证识别精度的同时，
兼顾模型的训练成本与推理效率，
是鸟类识别任务中的重要问题。

本项目以“**基于深度学习的鸟类识别小程序**”为研究目标，
探索利用卷积神经网络与迁移学习方法，
实现高效、可部署的鸟类物种识别模型，
并为后续的小程序端应用提供模型支持。

---

## 2. 项目目标

本项目的主要目标包括：

1. 基于公开鸟类数据集构建完整的鸟类图像分类流程；
2. 采用预训练卷积神经网络进行迁移学习，
   实现对鸟类物种的自动识别；
3. 对比不同训练策略（冻结骨干网络与全模型微调）
   在识别精度与训练效率上的差异；
4. 设计结构清晰、可复现实验过程的模型训练代码；
5. 为鸟类识别小程序提供可用的模型方案与实验依据。

---

## 3. 技术路线概述

本项目整体技术路线如下：

1. **数据准备阶段**
   - 选用 CUB-200-2011 鸟类数据集；
   - 按官方划分进行训练集与测试集划分；
   - 使用 `torchvision.datasets.ImageFolder` 进行数据加载；
   - 对输入图像进行统一尺寸调整与基本数据增强。

2. **模型设计阶段**
   - 选用 ResNet18 作为基础网络结构；
   - 使用 ImageNet 预训练权重进行初始化；
   - 将最后的全连接层替换为适用于 200 类鸟类分类的输出层；
   - 说明：ResNet18 本身属于卷积神经网络（CNN）的一种典型结构。

3. **训练策略设计**
   - 冻结骨干网络（Frozen Backbone）：
     仅训练最后的分类层，用作基线实验；
   - 全模型微调（Full Fine-tuning）：
     解冻全部网络参数，提高模型对细粒度特征的适应能力；
   - 根据训练策略调整学习率等超参数。

4. **实验与评估**
   - 在本地 GPU（RTX 3050）环境下完成主要实验；
   - 记录训练损失与验证集 Top-1 Accuracy；
   - 对比不同训练策略下模型性能的变化；
   - 为后续服务器 GPU 实验预留扩展空间。

## 4. 项目可行性分析

- **数据可行性**：
  CUB-200-2011 为成熟公开数据集，数据标注规范，适合学术研究与教学实验。

- **技术可行性**：
  ResNet18 网络结构成熟、稳定，
  计算资源需求适中，适合本科阶段实验与模型调试。

- **实现可行性**：
  项目基于 PyTorch 框架实现，
  代码结构清晰，实验过程可复现，
  具备在本地与服务器环境中运行的条件。

---

## 5. 项目预期成果

- 完整的鸟类识别模型训练代码；
- 冻结与全微调两种训练策略的实验结果对比；
- 清晰记录的实验日志与参数配置；
- 可作为鸟类识别小程序后端模型的基础方案；
- 为后续模型优化与功能扩展提供实验依据。
