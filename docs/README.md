# Docs 说明文档（Documentation）

本文件夹用于存放本项目的**设计说明与文档性材料**，  
主要用于对鸟类识别小程序的整体设计思路、数据集说明、
模型结构设计以及项目背景进行系统性描述。

该目录侧重于**“为什么这样设计”与“整体方案说明”**，
不直接记录具体的实验日志与训练过程细节。

---

## 文档结构说明

### 1. `project_overview.md`
项目总体说明文档，用于介绍本课题的研究背景、研究目标、
应用场景以及整体技术方案概述，帮助读者快速理解项目内容。

主要包括：
- 研究背景与意义  
- 项目目标与功能定位  
- 整体技术思路概述  
- 项目最终应用形式（鸟类识别小程序）

---

### 2. `dataset_description.md`
数据集说明文档，用于介绍本项目所使用的数据集来源、
数据规模及其在鸟类识别任务中的特点。

主要包括：
- 数据集名称与来源  
- 数据规模与类别数量  
- 数据集在细粒度图像分类中的应用价值  

---

### 3. `model_design.md`
模型设计说明文档，用于详细描述本项目所采用的深度学习模型结构、
设计思路以及训练策略。

主要包括：
- 模型总体设计思路  
- ResNet18 网络结构说明  
- 迁移学习策略（冻结骨干 / 全模型微调）  
- 模型选择理由与优缺点分析  

---

## 说明

- 本目录内容偏重**理论设计与方案说明**，用于论文写作与开题报告支撑。
- 具体实验过程、训练结果与对比分析记录在项目根目录下的
  `experiments/` 文件夹中。
- 文档内容以**中文为主，保留必要的英文专业术语**，便于指导教师阅读与理解。
